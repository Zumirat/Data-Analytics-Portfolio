### **Factory Device Health Analysis Dashboard**



Overview

This Tableau project analyzes unhealthy device counts across factories and device types.



The goal is to identify factories and device categories contributing most to device health issues.



Tools Used

\- Tableau Public

\- Data Visualization

\- Dashboard Design



Visualizations



Factory Analysis

Displays unhealthy device counts by factory location:

\- Daikibo Berlin

\- Daikibo Factory Meiyo

\- Daikibo Factory Seiko

\- Daikibo Shenzhen



Device Type Analysis

Displays unhealthy device counts across various device categories.



Key Insights

\- Daikibo Factory Seiko recorded the highest number of unhealthy devices.

\- Daikibo Shenzhen also showed a high concentration of unhealthy devices.

\- Certain device types contribute disproportionately to health issues.

\- Maintenance efforts should prioritize factories with the highest unhealthy device counts.
