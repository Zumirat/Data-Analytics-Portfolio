### Superstore Customer Analysis Dashboard



Overview

This project is an interactive Excel dashboard built to analyze customer demographics and distribution across different cities.



The dashboard provides insights into customer reach, age groups, gender distribution, and city-level customer concentration.



Tools Used

\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Slicers

\- Dashboard Design



Dashboard Features



KPI Cards

\- Total Reach

\- Average Age

\- Number of Cities

\- Female Customer Percentage

\- Male Customer Percentage

\- Non-binary Customer Percentage



Visualizations

\- Customers by City

\- Gender Distribution

\- Gender Breakdown by Age Group



Interactive Filters

\- Age Group

\- City

\- Gender



Key Insights

\- Faro recorded the highest customer count.

\- Porto had the lowest customer count among the selected cities.

\- Customer distribution is relatively balanced across gender categories.

\- The 46+ age group contains the highest number of customers.
